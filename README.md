# Stepper-28BYJ48
本程序基于Arduino UNO R3控制ULN2003驱动模块驱动五线四相八拍电机步进电机（28BYJ48）；除了驱动程序外，并且记录步数到EEPROM中用以在意外掉电重启后恢复之初始设定的位置。
